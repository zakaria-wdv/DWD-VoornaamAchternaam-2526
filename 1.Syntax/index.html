Console opmaak

Doel: console-berichten opmaken met kleuren en lettergrootte.

Theorie

Je kunt console-berichten opmaken met CSS door %c te gebruiken in combinatie met een CSS-string als tweede argument:

Je kunt ook meerdere %c placeholders gebruiken om verschillende delen verschillende stijlen te geven:

console.log('%cRood %cBlauw', 'color: red;', 'color: blue;');

String interpolatie

Array methodes

Doel: arrays transformeren, filteren en zoeken met map(), filter(), find() en includes().

Theorie

Voorbeelden van array methodes. Met includes(waarde) controleer je of de array een bepaalde waarde bevat (geeft true of false). 

const getallen = [1, 2, 3, 4, 5];

// includes: bevat een waarde?
getallen.includes(3);   // true ‚Äì array bevat 3
getallen.includes(99);  // false

// map: zet alle elementen om naar een nieuwe waarde
const verdubbeld = getallen.map(g => g * 2);
console.log(verdubbeld); // [2, 4, 6, 8, 10]

// filter: selecteer elementen op basis van een voorwaarde
const even = getallen.filter(g => g % 2 === 0);
console.log(even); // [2, 4]

// find: geef eerste element terug dat voldoet
const eersteGroterDanDrie = getallen.find(g => g > 3);
console.log(eersteGroterDanDrie); // 4

Iteraties

Doel: herhalingen schrijven met for, while en do-while.

Theorie

Herhalingen kun je op verschillende manieren schrijven. for: teller met start, conditie en stap. while: herhaal zolang de conditie waar is. do-while: minstens √©√©n keer uitvoeren, daarna conditie. 

// for: teller
for (let i = 0; i < 3; i++) {
   console.log(i); // 0, 1, 2
}

// while: zolang conditie waar is
let n = 0;
while (n < 2) {
   console.log(n);
   n++;
}

// do-while: minstens 1x
let m = 0;
do {
   console.log(m);
   m++;
} while (m < 2);

Opdracht

  1. Gebruik een for-loop om de getallen 1 tot en met 5 te loggen.
  2. Gebruik een do-while-loop vanaf 1 om het eerste getal te vinden waarvan de derde macht groter is dan 100.

For-of en forEach

Doel: door arrays itereren met for-of en forEach.

Theorie

for-of loopt over de waarden in een array (of andere iterable). forEach is een array-methode die een functie per element aanroept; de functie kan ook de index als tweede parameter krijgen. 

const arr = ['a', 'b', 'c'];

// for-of: over waarden
for (const el of arr) {
   console.log(el); // a, b, c
}

// forEach: methode op array (met index)
arr.forEach((el, index) => {
	console.log(`${index + 1}. ${el}`)
});

Opdracht

  1. Maak een array vakken met ['HTML', 'CSS', 'JavaScript'].
  2. Log elk vak met een for-of loop.
  3. Gebruik forEach op vakken om elk vak met volgnummer te loggen (bv. 1. HTML).

Spread operator

Doel: de ... spread operator gebruiken om waarden uit te spreiden tot een array.

Theorie

Met de ... spread operator kun je waarden uitspreiden de elementen van een array. Er zijn veel toepassingen; we bekijken er drie: 

// array samenvoegen
const arr1 = [1, 2, 3];
const arr2 = [4, 5];
const samengevoegd = [...arr1, ...arr2]; // [1, 2, 3, 4, 5]

// array kopi√´ren
const a = [1, 2, 3];
const b = [...a]; // maak kopie
b.push(4);
console.log(a); // [1, 2, 3]
console.log(b); // [1, 2, 3, 4]

// string omzetten naar array
const word = 'code';
const letters = [...word];
console.log(letters); // ['c', 'o', 'd', 'e']
